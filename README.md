# 🐍 Python Data Analysis Projects

A collection of Python projects focused on **data analysis, EDA, and visualization** — built to demonstrate real-world analytical thinking using Python.

---

## 📁 Projects

### 1. 📊 Customer Sales Analysis
**File:** `customer_sales_analysis.py`

Analyzed 500+ customer orders to uncover revenue trends, top-performing categories, and regional sales patterns.

- Performed data cleaning (null handling, type validation)
- Conducted EDA: revenue by category, region, and monthly trends
- Built a 4-panel dashboard with bar charts, pie chart, trend line, and histogram
- **Tech:** Python, Pandas, NumPy, Matplotlib, Seaborn

---

### 2. 🎓 Student Performance Analysis
**File:** `student_performance_analysis.py`

Explored academic performance data of 400 students to identify key factors affecting scores.

- Analyzed impact of study hours and attendance on exam scores
- Compared subject-wise performance across genders
- Visualized grade distribution and correlation trends
- **Tech:** Python, Pandas, NumPy, Matplotlib, Seaborn

---

### 3. 🌍 COVID-19 Data Analysis
**File:** `covid19_data_analysis.py`

Analyzed COVID-19 trends across 8 countries over 2021 — including new cases, fatality rates, and recovery patterns.

- Tracked weekly case trends and identified wave patterns
- Compared fatality rates and recovery rates by country
- Built a 4-panel visualization dashboard
- **Tech:** Python, Pandas, NumPy, Matplotlib, Seaborn

---

### 4. 🚦 Traffic Light Conditions
**File:** `Traffic Light Conditions.py`

Simulates traffic light logic using Python conditionals.
- **Concepts:** if-elif-else, user input handling

---

### 5. 📒 Contact Book
**File:** `contact book.py`

A CLI-based contact management app — add, search, and display contacts.
- **Concepts:** Dictionaries, functions, loops
